# Hi there 👋

我是你们的**凤鸣**，现就读于山东省枣庄市的枣庄市第三中学，目前是个高一学生

曾经有很多人问我为什么叫凤鸣，在此我解释一下。我第一次学棋是在2014年，棋院名字就叫**凤鸣棋院**；我的小学**枣庄市实验学校**在2016年和2017年时学校的报刊就叫**凤鸣报**。我把**凤鸣**当成自己的昵称也算是对儿时的一种怀念吧。

日常生活中我挺喜欢去全国各地旅游，也喜欢弹吉他，当然更喜欢化学这门神奇的学科（实际上一定程度也是因为某**M42**学姐才喜欢的）

嗯...回归正题。好吧，实际上这么多年**我从来没有专门机构学过编程**，仅仅是因为自己喜欢围棋且偶然接触到了围棋AI才自学的（读过我写的代码的都知道）

至于写什么呢？我个人更加倾向于创作**实用性的小程序**，这也与我的经历有关。2022年3月18日我初次接触围棋AI，不过那时候的资源很匮乏，使用AI等操作对我那样的新手小白来说简直难如登天，所以后期我始终在弥补这个空白，写了许多教程，方便后人更方便地使用

我自己的编程知识并不丰富，只会点简单的**Python**、**powershell**、**Jupyter Notebook**等，想学下Java结果一直没时间

个人主要在搞后端，对前端GUI以及网站什么的一窍不通
